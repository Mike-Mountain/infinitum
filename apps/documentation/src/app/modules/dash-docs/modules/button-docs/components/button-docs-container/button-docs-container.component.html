The DashButtonModule should be imported into whichever module needs it. If it is needed in more than one module, consider creating a dedicated library module in your application to import library components.

The dash-button wraps the Angular material button, enforcing only the style necessary for the Dashboard UI.

Models:

color: ButtonColor

Used to determine the button color. Accepts 'primary', 'accent', and 'warn'. Defaults to 'primary'.

type: ButtonType

The type of button to display. Accepts: 'basic' and 'icon'. Defaults to 'basic'.

role: ButtonRole

Whether the component should render an {{'<'}}a{{'>'}} or {{'<'}}button{{'>'}} element. Accepts 'link' and 'button'. Defaults to 'button'.

class: string

Used to add a global css class to the element. This class can only be styled from the global stylesheet.

style: {{'{'}}key, value{{'}'}}

Used to add inline style to the button element. Should be used only in extreme cases.
